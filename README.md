# Financial Management System

A web-based platform for managing personal and small business finances. Track income, expenses, and savings with dashboards, history, and analytics for both user types.

## Features
- Personal user and small business owner support
- Registration, login, and secure authentication
- Dashboard for income, expenses, and savings
- Add, edit, delete, and view history for expenses and savings
- Service selection and plan management
- Responsive UI with modern design
- Built with PHP, MySQL, HTML, CSS, JS

## Structure
- `layouts/`: Shared site views, controllers, models, assets
- `personal_user/`: Personal user features (dashboard, expenses, savings, registration, settings)
- `small_business_owner/`: Small business features (dashboard, expenses, savings, registration, settings)
- `financialmanagementsystem.sql`: Database schema and sample data

## Getting Started
1. Import `financialmanagementsystem.sql` into your MySQL/MariaDB server
2. Configure your web server to serve the project directory
3. Update database connection settings in model files if needed
4. Access the app via browser
